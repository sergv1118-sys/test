# GlobeNewswire Real-Time News Monitor

Мониторинг новостей и пресс-релизов компаний с сайта GlobeNewswire.com в реальном времени.

## Описание

Скрипт автоматически собирает новости о публичных компаниях, торгующихся на бирже, извлекает тикеры компаний и сохраняет новости в файлы, названные по тикеру компании.

## Возможности

- **Автоматический сбор новостей** - мониторинг нескольких категорий новостей
- **Извлечение тикеров** - автоматическое распознавание тикеров NYSE, NASDAQ из текста новостей
- **Сохранение по тикерам** - каждая новость сохраняется в файл с именем тикера (например, `AAPL.txt`, `GOOGL.txt`)
- **Режим реального времени** - периодическая проверка новых новостей
- **Поиск по компании** - возможность поиска новостей по названию компании
- **Кэширование** - запоминание обработанных новостей для избежания дубликатов

## Установка зависимостей

```bash
pip install requests beautifulsoup4 lxml schedule
```

## Использование

### Однократный запуск (проверить новости один раз)

```bash
python globenewswire_monitor.py --once
```

### Непрерывный мониторинг (каждые 5 минут)

```bash
python globenewswire_monitor.py
```

### Непрерывный мониторинг с другим интервалом

```bash
python globenewswire_monitor.py --interval 10
```

### Поиск новостей о конкретной компании

```bash
python globenewswire_monitor.py --search "Apple Inc"
```

### Указание своей директории для вывода

```bash
python globenewswire_monitor.py --once --output-dir ./my_news
```

### Мониторинг конкретных категорий

```bash
python globenewswire_monitor.py --once --categories /news/energy /news/technology
```

## Структура выходных файлов

Каждая новость сохраняется в формате:

```
================================================================================
DATE: March 06, 2026 16:01 ET|Source:Company Name
TITLE: Заголовок новости
URL: https://www.globenewswire.com/news-release/...
TICKERS: AAPL, GOOGL
SCRAPED AT: 2026-04-03T11:11:45.354950
================================================================================

Текст новости...

================================================================================
```

## Конфигурация

Основные параметры можно изменить в начале скрипта в словаре `CONFIG`:

- `base_url` - базовый URL GlobeNewswire
- `news_categories` - список категорий новостей для мониторинга
- `output_dir` - директория для сохранения файлов
- `check_interval_minutes` - интервал проверки (для непрерывного режима)
- `max_news_per_run` - максимальное количество новостей за один проход

## Логирование

Все действия скрипта логируются в файл `news_monitor.log` и выводятся в консоль.

## Доступные категории новостей

- `/news/consumer-products-services` - Потребительские товары и услуги
- `/news/banks-financial-services` - Банки и финансовые услуги
- `/news/energy` - Энергетика
- `/news/heathcare` - Здравоохранение
- `/news/industrials-utilities` - Промышленность и коммунальные услуги
- `/news/technology` - Технологии

## Пример работы

После запуска скрипта в директории `news_output` появятся файлы:

```
news_output/
├── AAPL.txt          # Новости Apple
├── GOOGL.txt         # Новости Google
├── MSFT.txt          # Новости Microsoft
├── UNKNOWN.txt       # Новости без распознанных тикеров
└── processed_urls.json  # Кэш обработанных URL
```

## Примечания

- Скрипт автоматически определяет тикеры компаний из текста новостей
- Новости без распознанных тикеров сохраняются в файл `UNKNOWN.txt`
- При повторном запуске скрипт не обрабатывает уже полученные новости
- Для корректной работы требуется стабильное интернет-соединение
